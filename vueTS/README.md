# my-api-demo

To demo how generated client APIs could be compiled and run together with Vue TS codes.

0: npm install -g @vue/cli
0.1: npm i
1: start Web API
2: npm test

## Project setup
```
npm install
```

### Compiles and hot-reloads for development

npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests

npm run test

```
